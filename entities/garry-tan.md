# Garry Tan

**Role**: President & CEO, Y Combinator
**Known for**: GBrain (open-source AI agent memory system), YC leadership, startup investing
**GitHub**: [garrytan](https://github.com/garrytan)

## Key Contributions to Agent Memory

- Built **GBrain** — production brain running 146,646 pages, 24,585 people, 5,339 companies, 66 cron jobs autonomously
- Pioneer of the "markdown-first" brain pattern: brain repo is plain Markdown in git, diffable, branchable
- Designed compounding memory: tiered enrichment + fail-improve loop + backlink-boosted ranking
- Zero-LLM-call entity extraction architectural pattern — keeps ingestion at $0 token cost
- Authored the "cerebellum vs prefrontal cortex" framework: offload routine tasks to reflex so complex thinking has a clean foundation

## Philosophy

> "Search gives you raw pages. GBrain gives you the answer."

Believes agent memory should compound over months, not deliver value in a one-week trial. The point of a 100K-page brain is to use it as a **strategic moat** — never lose context, query what's in your own head without re-reading it.

## Cerebellum Framework

Most agent builders focus on the prefrontal cortex (planning, reasoning, multi-step chains — "CEO-stuff"). Tan argues the real value is building the **cerebellum** — offloading boring tasks into reflex so complex thought can happen on a clean foundation.
