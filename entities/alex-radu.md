# Alex Radu

**Role**: Founder, UBUNTU Nation
**Age**: 37
**Timezone**: London (GMT/BST)
**Background**: Entrepreneur

## Personal Motivation

Wants to build projects that bring freedom for communities, individuals, and all humanity. Sees UBUNTU Nation as his life's mission to help change the world, help humanity evolve, and help individuals achieve their best possible potential.

> "UBUNTU Nation is my life's mission to help change the world for the better, to help humanity evolve and to help individuals achieve their best possible potential, for their quality of life and for the rest of society's betterment."

## Working Style

- **Non-programmer** — uses Claude Code / cloud-based AI coding tools
- Conceptual understanding of HTML, Python, CSS, JavaScript, TypeScript — still learning
- Prefers **casual but elevated** conversation — intelligent, educated, concise, non-formal
- Wants to grow knowledge together with AI — mutual intelligence-building
- **No task runs without confirmation** — Alex must be consulted before any work begins; time/token estimates provided for approval

## Project Vision

Building a sovereign, AI-native global society:
- Physical villages (Tenerife first, then El Salvador, then global)
- Bitcoin-native treasury
- On-chain governance
- AI embedded in all operations — content, strategy, infrastructure, community
- Target: 5,000 waitlist signups

## Communication Preferences

- Casual, intelligent, educated but not formal
- Truthful and concise
- First principles thinking — break problems to fundamental truths, reject reasoning by analogy
- Brave Search preferred for web research
